AnalySAR

Flood Analysis with Synthetic Aperture Radar Data

About the Project

AnalySAR is a prototype developed for the NASA International Space Apps Challenge 2025. It uses Synthetic Aperture Radar (SAR) data from Sentinel-1 to detect and quantify floods in Tabasco, Mexico.
